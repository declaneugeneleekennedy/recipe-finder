# RecipeFinder

## Requires
- NodeJS v0.12.2

## Install
1.  Open the project folder in a terminal
2.  run:

    `npm install`

## Running the App
1.  Open the project folder in a terminal
2.  run:

    `node index --fridge=/path/to/fridge.csv --recipes=/path/to/recipes.json`

## Running Tests
1.  Open the project folder in a terminal
2.  run:

    `node test`
